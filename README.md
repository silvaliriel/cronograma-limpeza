# 🧹 Cronograma de Limpeza

App de cronograma de limpeza para lar com pets, otimizado para celular.

## 🚀 Como publicar no GitHub Pages (passo a passo)

### 1. Criar o repositório
1. Acesse [github.com](https://github.com) e faça login (ou crie uma conta grátis)
2. Clique no **+** no canto superior direito → **New repository**
3. Dê o nome: `cronograma-limpeza`
4. Deixe como **Public**
5. Clique em **Create repository**

### 2. Subir o arquivo
1. Na página do repositório, clique em **uploading an existing file**
2. Arraste o arquivo `index.html` para a área indicada
3. Clique em **Commit changes**

### 3. Ativar o GitHub Pages
1. Vá em **Settings** (engrenagem) → **Pages** no menu lateral
2. Em **Source**, selecione **Deploy from a branch**
3. Em **Branch**, selecione **main** e pasta **/ (root)**
4. Clique em **Save**
5. Aguarde 1-2 minutos ⏳

### 4. Acessar o app
Seu app estará disponível em:
```
https://SEU_USUARIO.github.io/cronograma-limpeza/
```

## 📱 Adicionar à tela inicial do celular

### Android (Chrome)
1. Abra o link no Chrome
2. Toque nos **3 pontinhos** (⋮) no canto superior direito
3. Toque em **"Adicionar à tela inicial"**
4. Confirme → o ícone aparece na sua tela! ✅

### iPhone (Safari)
1. Abra o link no **Safari** (não funciona no Chrome do iOS)
2. Toque no ícone de **compartilhar** (□↑) na barra inferior
3. Role e toque em **"Adicionar à Tela de Início"**
4. Confirme → o ícone aparece na sua tela! ✅

## 💾 Sobre o salvamento dos dados

Os checks ficam salvos no **armazenamento local do celular** (localStorage).  
Isso significa:
- ✅ Fecha o app → abre de novo → tudo salvo
- ✅ Sem internet → funciona normalmente  
- ⚠️ Se limpar os dados do navegador → os checks são apagados
- ⚠️ Não sincroniza entre dispositivos diferentes

## 📋 Funcionalidades

- ☀️ **Rotina Diária** — 5 tarefas × 6 dias (Seg a Sáb)
- 📅 **Semanal** — Grade com detalhes de cada dia + Semana 1 a 5
- 📆 **Quinzenal** — Mai a Dez com 2 campos por mês
- 🗓️ **Mensal** — Mai a Dez
- 💡 **Dicas** — Pets e Lavanderia
- 📊 Progresso visual em cada seção
- 🔄 Botão para zerar tudo
